This project focuses on analyzing and visualizing coffee shop sales data using Excel. The goal was to transform raw transactional data into actionable insights for business decision-making.

- Collected and cleaned raw transactional data to ensure accuracy and consistency for analysis.
- Designed an interactive Excel dashboard to track key business metrics including:

- **Excel:** Data cleaning, calculations, pivot tables, charts, and dashboard design
- **Data Visualization:** Dynamic charts, slicers, and KPI tracking
- **Business Intelligence:** Sales performance analysis and insights generation
